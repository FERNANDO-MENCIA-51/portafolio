Proyecto de Automatizacion

Desarrollo de un sistema de automatización utilizando Google Apps Script y las herramientas de Google Workspace. El proyecto integra Google Forms para la recolección de datos, Google Sheets como base de datos, y Google Sites para la interfaz de usuario. Implementé scripts personalizados para automatizar flujos de trabajo, sincronización de datos entre hojas de cálculo, generación automática de documentos en Google Drive, y envío de notificaciones por correo. El sistema permite una gestión eficiente de la información con almacenamiento seguro en la nube y acceso desde cualquier dispositivo.
